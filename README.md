# IV Checker For Brain
## 概要
SHARP製電子辞書「Brain」用のポケモンGOのPvP（対人戦）における個体値ランク（SCPランク）を計算するPythonベースのアプリです。CPUが非力かつメモリが少ない電子辞書向けに最適化しています。もちろん、app/\_\_main__.pyを実行すれば一般的なPCのPython環境でも動作します。
[python-app-template](https://github.com/brain-hackers/python-app-template.git)を使用しています。

## 導入方法
※日本語を表示するためにまず[ceOpener](https://brain.fandom.com/ja/wiki/CeOpener)を入れておく必要があります。

まずGitHub Actionsでbuild.ymlを実行し、終わるまで待ちArtifactsのiv_checker.zipをダウンロードし解凍します。
その後AppMain.exeがあるフォルダをSDカードまたは本体メモリの「アプリ」フォルダ内に入れてください。
